Privacy Policy

Effective date: April 12, 2026

What Better Bear does

Better Bear is an open-source MCP server and CLI that lets you access your Bear notes from Claude Desktop, Claude Code, or the terminal. It communicates with Apple's CloudKit service using your own iCloud credentials.

Data collection

Better Bear does not collect, transmit, or store any personal data on external servers. Specifically:

  • No analytics or telemetry
  • No crash reporting to third parties
  • No advertising or tracking
  • No data shared with the developer or any third party

Data that stays on your machine

The following data is stored locally on your computer and never leaves it:

  • iCloud authentication tokens — stored in ~/.config/bear-cli/auth.json. These are your Apple CloudKit session tokens, used to access your iCloud private database. They are never transmitted anywhere except to Apple's CloudKit API.
  • Note cache — a local cache of your Bear notes in ~/.config/bear-cli/ to avoid re-downloading unchanged notes on every request.
  • Context library — if you use the context library feature, a folder of markdown files synced from your Bear notes is stored locally (default: ~/.bear-context/).

Network requests

Better Bear makes network requests only to:

  • Apple CloudKit (api.apple-cloudkit.com) — to read, create, edit, and sync your Bear notes. This is the same API used by Bear Web.
  • GitHub (api.github.com) — only when checking for updates via bcli upgrade.
  • npm registry (registry.npmjs.org) — only when Claude Desktop or Claude Code launches the MCP server via npx.

No note content, metadata, or personal information is sent anywhere else.

MCP server communication

When used as an MCP server, Better Bear communicates with the MCP client (Claude Desktop or Claude Code) over local stdio pipes. Note content is passed between the MCP server process and the client process on your machine. This communication does not leave your computer — the MCP client may send note content to Anthropic's API as part of the conversation context, subject to Anthropic's own privacy policy.

Third-party services

  • Apple iCloud — your notes are stored in your iCloud private database, governed by Apple's Privacy Policy.
  • Anthropic (Claude) — if you use Better Bear with Claude Desktop or Claude Code, note content shared during conversations is subject to Anthropic's Privacy Policy.

Data deletion

To remove all locally stored data:

  • Delete ~/.config/bear-cli/ (auth tokens and cache)
  • Delete ~/.bear-context/ (context library, if used)
  • Run bcli mcp uninstall to remove MCP server configuration

Open source

Better Bear is fully open source under the MIT license. You can audit the code at github.com/KuvopLLC/better-bear.

Changes to this policy

If this policy changes, the updated version will be posted at this URL. Better Bear does not collect email addresses, so there is no way to notify users directly — check this page if you have concerns.
