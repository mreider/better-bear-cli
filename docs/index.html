better-bear-cli

A CLI and MCP server for Bear notes. Talk to your notes from the terminal or let AI assistants read, search, and edit them via CloudKit.

Download GitHub
curl -sL https://raw.githubusercontent.com/mreider/better-bear-cli/main/install.sh | bash

CLI

Create, search, edit, tag, and attach files to Bear notes from the terminal. Works without Bear open.

MCP Server

19 tools for Claude Desktop. Let AI read, write, and organize your Bear notes directly.

CloudKit API

Same REST API Bear Web uses. No SQLite hacking, no x-callback-url. Syncs to all devices.

Open Source

MIT licensed. Contributions welcome. Buy me a coffee if you find it useful.

Commands

bcli lsList notes
bcli search "query"Full-text search with date filters
bcli get <id>View a note (includes parsed front matter)
bcli create "Title"Create a note with tags and front matter
bcli edit <id>Append, replace, or edit sections and front matter
bcli attach <id> fileUpload images and files to iCloud
bcli tag add/remove/rename/deleteManage tags across notes
bcli archive <id>Archive or unarchive
bcli trash <id>Move to trash
bcli todoList and toggle TODO items
bcli export ./dirExport notes as markdown
bcli upgradeSelf-update to latest release

MCP Server

Install the MCP server for Claude Desktop:

cd mcp-server && npm install && npm run build && cd .. bcli mcp install

19 tools including notes, tags, search, attachments, TODOs, front matter, and archive. Restart Claude Desktop after installing.
